安装 [木兰](https://pypi.org/project/ulang/) 后在根目录下运行:

- 同步 `仓库列表.csv` 中所有 git 仓库内容，新仓库则 clone，已有仓库则 pull。如 pull 报错，则备份后再 clone。
```
$ 木兰 原型/git同步.ul
```

- `统计.ul` 用于列出每个博客的简报：所在博客，文件个数, 所占空间, 更新时间, 最新博文
- `列表.ul` 列出所有博文目录：所在博客，时间戳，题目
